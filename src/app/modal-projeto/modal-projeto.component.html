Aion
A rede social Aion surgiu como projeto integrador durante o Bootcamp da Generation Brasil em 2022, baseando-se na ODS 8 da ONU: Desenvolvimento Sustentável, Trabalho Descente e Crescimento Econômico. Surgimos com propósito de aumentar a taxa de reciclagem de lixos eletrônicos promovendo uma rede consciente e descontraída de conversas. O Back-end foi feito com o framework SpringBoot, com o banco de dados MySQL e os testes de aplicação pelo JUnit e Postman. No Front-end consumimos a API através do framework Angular com a linguagem Typescript. O Deploy do back-end foi feito pelo Heroku, e o front-end pelo Netlify.
